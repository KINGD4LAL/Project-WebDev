Introduction

This unit introduces computer programming in a high-level programming language and includes principles and practice in problem-solving, program design, solution implementation and testing. You will gain practical experience in developing software using industry-standard programming languages and tools. This will act as an introduction to the basics of programming that you will need in order do well in your programming modules in the 1st year. It will cover the basic concepts of variables and data types, arrays, and functions.
Variables and data types:
A variable is a packet of data the store a certain value. Much like in algebra where we use symbols to represent numbers such as x=5, in this case the variable is named ‘x’ and it stores the value 5.
The java for this is: int x = 5;
We can use this variable later in the program in place of its value, for example: “Bobby is x years old”. This will be read by the program as “Bobby is 5 years old”. Variables aren’t just used so to save writing out the same value again, they can also be changed. In a years time Bobby will be 6, so we have to update x to be 6.
The java for this is: x = 5;
The program will now read the sentence “Bobby is x years old” as “Bobby is 6 years old”. We can also use variables for calculations, such as x+5, and we get an answer of 10.
It is not just numbers we can store in variables. The contents of the variable is defined by its data type. For x this was ‘int’ short for integer meaning at can store whole numbers. The most common data types you will be using are:
• int: Stores whole numbers from -2,147,483,648 to 2,147,483,647
• float: Stores fractional numbers. Sufficient for storing 6 to 7 decimal digits
• string: Strings are used for storing text
• boolean: Stores true or false values
• char: Stores a single character/letter or ASCII values
The java for creating a variable is, Datatype Name; such as Int Age;. When creating a variable you can also give it value for that we do, Datatype Name = Value; such as Int Age = 20;. We can also change the value of a variable with the code Name = Value; such as Age = 21. However we need to assign a value that matches the data type in this case an int, we could not assign the value “Twenty” as that would be a string.

Sets

Logic

Graph Theory
