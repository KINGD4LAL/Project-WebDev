Software Testing and Quality Assurance

Introduction

In this unit, you will study a variety of contemporary software verification and testing tools and techniques. You will gain experience testing various aspects of a systems behaviour using industry-standard test automation tools and strategies. Software Testing & Quality Assurance is also going to be about how we go about finding bugs in software. It will cover various testing approaches and tools, covering several languages.

Programming Languages and Paradigms

Introduction

This unit examines the design and evolution of programming languages with a focus on comparative analysis and critical evaluation of programming language concepts and paradigms.Programming Languages & Paradigms gives you some breadth of programming experience, trialling a few new languages that work in different ways to what you're used to. Synoptic Project

Introduction

The synoptic project allows you to consolidate your learning from your degree programme into a final project, bringing together ideas and techniques from throughout your study. From this project, you will produce the centrepiece of their degree work portfolio. will involve knowledge and skills from across the course. You'll get a short (2-3 paragraph) project brief, and have to develop the idea into a proper finished product, negotiating with your supervisor over how things should work, how it should be designed etc.

         

Cloud Computing

Introduction

You will architect scalable software solutions using commercially available cloud infrastructure. You will learn to make informed decisions on which cloud provider is most suitable for a project based on cost or technical suitability. In addition, you will gain experience of developing software using the tools and published documentation of a platform provider. This will involve some study of the virtualisation technologies that make the cloud work, as well as some development of software using a commercial cloud service platform using their APIs and tools.

Optional Unit

Introduction

In this Unit you will have the chance to pick one of Four Topics listed:
1. Mobile Computing
This unit offers you the opportunity to explore the development of applications designed to run on mobile computing devices. You will gain practical experience of using device emulators, frameworks, APIs and libraries.
2. Rapid Applied Problem Solving
This unit challenges you to develop your problem-solving skills under pressure. By participating in a series of rapid, applied problem solving challenges.
3. Research in Computing
The Research in Computing option cluster offers you the opportunity to engage with the Department of Computing and Mathematics Centre for Advanced Computational Science, by choosing an option which relates to one of the centres main themes.
4. User Experience and Interaction Design
Learn about the standard ways of assessing how usable user interfaces are, and about the standards and guidelines developers use to make better UIs
